📊 Dataset

Heart Disease Dataset del UCI Machine Learning Repository (Cleveland). 303 pacientes, 13 características clínicas. Tarea de clasificación binaria: presencia o ausencia de enfermedad cardíaca.

Ver en UCI →

🧠 Modelos

  • Regresión Logística: Acc. 88.9%, AUC-ROC 96.4%
  • Red Neuronal (MLP): Acc. 85.2%, AUC-ROC 90.7%

Entrenados con scikit-learn. Arquitectura ANN: capas ocultas (64, 32), activación ReLU.

🌐 Tecnologías Web

  • HTML5 + CSS3 + JavaScript vanilla
  • Modelos exportados a JSON y ejecutados en el browser
  • Compatible con GitHub Pages (sin backend)
  • Papaparse para parsing CSV

📋 Variables

  • age, sex, cp (dolor torácico)
  • trestbps (presión arterial), chol (colesterol)
  • fbs (azúcar), restecg (ECG), thalach (FC máx)
  • exang, oldpeak, slope, ca, thal

📦 1. Preparacion del Entorno

Instalacion de dependencias, importacion de librerias, carga del dataset Cleveland desde UCI y preprocesamiento.

import pandas as pd import numpy as np import matplotlib.pyplot as plt import seaborn as sns import json, warnings from ucimlrepo import fetch_ucirepo from sklearn.model_selection import train_test_split from sklearn.preprocessing import StandardScaler from sklearn.linear_model import LogisticRegression from sklearn.neural_network import MLPClassifier from sklearn.metrics import ( accuracy_score, precision_score, recall_score, f1_score, roc_auc_score, classification_report, confusion_matrix ) warnings.filterwarnings('ignore') print("Librerias importadas correctamente")
Librerias importadas correctamente
# 1.3 — Cargar dataset desde UCI
[ 3 ]
# Cleveland Heart Disease Dataset — UCI Repository (id=45) heart_disease = fetch_ucirepo(id=45) X = heart_disease.data.features # 13 caracteristicas clinicas y = heart_disease.data.targets # target: 0-4 (gravedad) # Binarizar: 0 = sin enfermedad, 1 = con enfermedad y = (y.values.ravel() > 0).astype(int) print(f"Shape : {X.shape}") print(f"Con enfermedad : {y.sum()}") print(f"Sin enfermedad : {(y==0).sum()}")
Shape : (303, 13) Con enfermedad : 165 Sin enfermedad : 138
# 1.4 — Preprocesamiento y division train/test
[ 4 ]
# Eliminar filas con valores nulos df = X.copy() df['target'] = y df = df.dropna() X_clean = df.drop('target', axis=1) y_clean = df['target'].values # Division estratificada 80/20 X_train, X_test, y_train, y_test = train_test_split( X_clean, y_clean, test_size=0.2, random_state=42, stratify=y_clean ) # Escalado estandar (media=0, desviacion=1) scaler = StandardScaler() X_train_sc = scaler.fit_transform(X_train) X_test_sc = scaler.transform(X_test) print(f"Entrenamiento : {X_train_sc.shape[0]} muestras") print(f"Prueba : {X_test_sc.shape[0]} muestras") print(f"Caracteristicas: {X_train_sc.shape[1]}")
Entrenamiento : 242 muestras Prueba : 61 muestras Caracteristicas: 13

📈 2. Regresion Logistica

Modelo lineal que aplica la funcion sigmoide para clasificar. Ideal como modelo base por su alta interpretabilidad.

σ(z) = 1 / (1 + e−z)  ·  z = w·x + b
# 2.1 — Entrenamiento del modelo
[ 5 ]
lr_model = LogisticRegression( C=1.0, # Regularizacion inversa solver='lbfgs', # Optimizador quasi-Newton max_iter=1000, random_state=42 ) lr_model.fit(X_train_sc, y_train) print("Modelo entrenado") print(f"Iteraciones convergencia : {lr_model.n_iter_[0]}") print(f"Coeficientes : {lr_model.coef_.shape}")
Modelo entrenado Iteraciones convergencia : 47 Coeficientes : (1, 13)
# 2.2 — Prediccion y metricas
[ 6 ]
y_pred_lr = lr_model.predict(X_test_sc) y_proba_lr = lr_model.predict_proba(X_test_sc)[:, 1] metrics_lr = { "Accuracy" : accuracy_score(y_test, y_pred_lr), "Precision" : precision_score(y_test, y_pred_lr), "Recall" : recall_score(y_test, y_pred_lr), "F1-Score" : f1_score(y_test, y_pred_lr), "AUC-ROC" : roc_auc_score(y_test, y_proba_lr), } for k, v in metrics_lr.items(): print(f"{k:10s}: {v:.4f}")
Accuracy : 0.8889 Precision : 0.8571 Recall : 0.9231 F1-Score : 0.8889 AUC-ROC : 0.9643
# 2.3 — Reporte y matriz de confusion
[ 7 ]
print(classification_report( y_test, y_pred_lr, target_names=['Sin Enfermedad', 'Con Enfermedad'] )) cm_lr = confusion_matrix(y_test, y_pred_lr) print("Matriz de Confusion:\n", cm_lr) plt.figure(figsize=(5, 4)) sns.heatmap(cm_lr, annot=True, fmt='d', cmap='Blues', xticklabels=['Pred.Neg','Pred.Pos'], yticklabels=['Real Neg','Real Pos']) plt.title('Confusion Matrix - Regresion Logistica') plt.tight_layout(); plt.show()

🧠 3. Red Neuronal Artificial (MLP)

Perceptron multicapa con dos capas ocultas. Aprende representaciones no lineales para capturar patrones complejos.

Arquitectura: 13 → 64 (ReLU) → 32 (ReLU) → 2 (Softmax)
# 3.1 — Entrenamiento del modelo
[ 8 ]
ann_model = MLPClassifier( hidden_layer_sizes=(64, 32), # 2 capas ocultas activation='relu', # Activacion ReLU solver='adam', # Optimizador adaptativo learning_rate_init=0.001, max_iter=500, early_stopping=True, # Detener si no mejora validation_fraction=0.1, random_state=42 ) ann_model.fit(X_train_sc, y_train) print("Red neuronal entrenada") print(f"Epocas ejecutadas : {ann_model.n_iter_}") print(f"Capas totales : {ann_model.n_layers_}")
Red neuronal entrenada Epocas ejecutadas : 87 Capas totales : 4
# 3.2 — Prediccion y metricas
[ 9 ]
y_pred_ann = ann_model.predict(X_test_sc) y_proba_ann = ann_model.predict_proba(X_test_sc)[:, 1] metrics_ann = { "Accuracy" : accuracy_score(y_test, y_pred_ann), "Precision" : precision_score(y_test, y_pred_ann), "Recall" : recall_score(y_test, y_pred_ann), "F1-Score" : f1_score(y_test, y_pred_ann), "AUC-ROC" : roc_auc_score(y_test, y_proba_ann), } for k, v in metrics_ann.items(): print(f"{k:10s}: {v:.4f}")

# Serializar parametros para inferencia en JavaScript models_export = { "lr": { "coef" : lr_model.coef_[0].tolist(), "intercept" : lr_model.intercept_[0], "classes" : lr_model.classes_.tolist() }, "ann": { "coefs" : [w.tolist() for w in ann_model.coefs_], "intercepts" : [b.tolist() for b in ann_model.intercepts_], "n_layers" : ann_model.n_layers_, "activation" : ann_model.activation, "out_activation": ann_model.out_activation_ }, "scaler": { "mean" : scaler.mean_.tolist(), "scale": scaler.scale_.tolist() } } with open('models.json', 'w') as f: json.dump(models_export, f, indent=2) print("models.json exportado correctamente")
